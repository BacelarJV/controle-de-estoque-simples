# 📦 Controle de Estoque Simples

## 📌 Visão Geral
Um sistema web para controle de estoque que permite cadastrar, atualizar e monitorar produtos de forma simples e eficiente.  
O sistema ajuda pequenos negócios a evitar perdas, controlar entradas e saídas e manter o estoque sempre atualizado.

## 🚀 Stack Tecnológica
- Node.js
- Express ou NestJS
- React
- PostgreSQL
