# NestJS Expert Skill

Use boas práticas do NestJS:
- arquitetura modular;
- controllers apenas para entrada HTTP;
- services para regra de negócio;
- DTOs para validação;
- filters para tratamento de exceções;
- interceptors para padronização de resposta;
- middlewares para tarefas globais como logging;
- evitar lógica de negócio em controllers.
